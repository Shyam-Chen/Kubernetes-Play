# Kubernetes Play

## Getting Started

Follow steps to execute this boilerplate.

1. Clone this boilerplate

```bash
$ git clone --depth 1 https://github.com/Shyam-Chen/Kubernetes-Play.git <PROJECT_NAME>
$ cd <PROJECT_NAME>
```

2. Apply the config to resources

```bash
$ kubectl apply -f kubernetes
```
